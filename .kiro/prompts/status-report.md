name: status-report
description: Get a status report of all open GitHub issues and team progress
---

Check all open GitHub issues for this repo and give me a status report:

1. **Done**: Issues closed since last check
2. **In Progress**: Issues with recent comments or assigned
3. **Blocked**: Issues labeled `blocked`
4. **Unassigned**: Issues nobody has picked up yet

Group by label (frontend, backend, infra). Flag anything that's been open too long without activity.
